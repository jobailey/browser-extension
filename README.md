# Browser Extension

## Purpose
**As of 3/19** This is a project intended to allow for me to brush up on HTML, JavaScript, and JSON by manipulating a browser extension.

## Usage
***Geared toward Mozilla Firefox browser settings***

- Open Microsoft Edge 
- Choose "More" icon
- Select extensions
